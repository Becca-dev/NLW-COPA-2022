# NLW-COPA-2022

Projeto com tema da copa do Mundo 2022 no Qatar, foi desenvolvido uma página web com o calendário dos jogos.

## Tecnologia utilizadas

HTML , CSS , JavaScript 

### Como foi feito 

Foi utilizado o HTML somente como estrutura base da página para estar recebendo o link do CSS para a parte de estilo e o script do para as funções do JavaScript para a criação dos cards onde vai receber do dados dos times de horários dos jogos.
